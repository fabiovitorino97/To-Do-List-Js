# To-Do List

Bem-vindo à To-Do List, uma aplicação web simples para gerenciar suas tarefas.

## Funcionalidades

- Adicione tarefas com prioridades.
- Organize as tarefas por urgência e importância.
- Exclua tarefas facilmente.
